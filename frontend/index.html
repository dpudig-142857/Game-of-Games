Game of Games

On the main page:

Start New Game


Create a new Game of Games (outlined below)

Continue Game


Continue an incomplete Game of Games

View Logs


View the log of any previous Game of Games

View Stats


View the overall stats of any player or any game

Starting a new Game of Games:

Step 1:


Select the options for the type of GoG that will be played

Step 2:


Select which players will be playing in the GoG

Step 3:


Select which games will be available to play in the GoG

Step 4:


Select the specialty game(s) for each player

Step 5:


Confirm the info and
begin the Game.
Good Luck!
